# Griffpatch Academy Hub Poll Extension

This browser extension adds a quick-access "Add Poll" button to the [Griffpatch Academy Hub](https://hub.griffpatch.academy/) discussion toolbar.

## Features
- Injects a bar-chart icon next to the "Add Video" button.
- Opens a community-made poll creator in a clean popup.
- Works on both Chrome and Firefox.

## Installation

### Chrome
1. Download this repo as a ZIP and extract it.
2. Go to `chrome://extensions/`.
3. Turn on **Developer Mode**.
4. Click **Load unpacked** and select the folder.

### Firefox
1. Go to `about:debugging`.
2. Click **This Firefox**.
3. Click **Load Temporary Add-on...** and select the `manifest.json`.
